# 📚 Learning Journal Reverse Engineering

Repository ini berisi dokumentasi proses pembelajaran mata kuliah **Reverse Engineering** selama satu semester. Setiap entri disusun sebagai catatan belajar mingguan yang merangkum materi yang dipelajari, konsep-konsep penting, refleksi pemahaman, serta referensi yang digunakan selama proses pembelajaran.

Tujuan dari repository ini adalah sebagai media dokumentasi pembelajaran sekaligus sarana untuk memperdalam pemahaman mengenai teknik dan metodologi Reverse Engineering, baik pada perangkat lunak (*software*) maupun perangkat keras (*hardware*).

---

## 📖 Isi Repository

Repository ini mencakup:

- 📝 Ringkasan materi setiap pertemuan
- 💡 Penjelasan konsep-konsep penting
- 🔍 Refleksi pembelajaran
